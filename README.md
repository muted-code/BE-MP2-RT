# Study Room - Backend Principal

API colaborativa en tiempo real para estudiantes.

## Requisitos
- Node.js (v16+)
- npm

## Instalacin
```bash
npm install
```

## Desarrollo
Para iniciar el servidor en modo desarrollo con recarga automtica:
```bash
npm run dev
```

## Documentacin API
Una vez que el servidor est corriendo, puedes ver la documentacin en:
[http://localhost:3001/api/docs](http://localhost:3001/api/docs)

## Despliegue
Para generar el build de produccin:
```bash
npm run build
```
Para iniciar en produccin:
```bash
npm start
```
