# Study Room - Backend de Tiempo Real

Esqueleto para el backend de tiempo real de la aplicación Study Room.

## Tecnologías
- Node.js
- TypeScript
- Express
- Socket.io

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Construcción (Build)
```bash
npm run build
```

## Producción
```bash
npm start
```

## Endpoints de la API
- **GET /api/health**: Verifica el estado del servicio.

## Eventos de Socket.io
- **ping**: Retorna `pong` con un mensaje.
