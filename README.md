# Study Room - Backend Realtime

Skeleton for the real-time backend of the Study Room application.

## Technologies
- Node.js
- TypeScript
- Express
- Socket.io

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Production
```bash
npm start
```

## API Endpoints
- **GET /api/health**: Check service status.

## Socket.io Events
- **ping**: Returns `pong` with a message.
